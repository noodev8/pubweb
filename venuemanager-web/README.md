# Venue Manager Web

Next.js admin UI for the Venue Manager.

## Setup (Coming Soon)

This project will be set up with:
```bash
npx create-next-app@latest . --typescript --tailwind --eslint --app --src-dir --import-alias "@/*" --use-npm
npx shadcn@latest init
```

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Shadcn/UI components

## Features (Planned)

- Venue information management
- Opening hours with visual week view
- Menu management with drag-and-drop
- Quick "86" toggle for items
- Room/accommodation management
- Attractions management
- Page content editing
